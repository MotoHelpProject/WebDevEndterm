# Project Name

## 📌 Description
1. MotoHelp: Социальный радар для байкеров
Это веб-приложение для взаимовыручки на дороге. Представьте: мотоциклист сломался на трассе или закончился бензин. Обычный эвакуатор — это дорого и долго.

Как это работает: Пользователь нажимает кнопку «Нужна помощь» и выбирает категорию (бензин, инструмент, эвакуация). Все райдеры в радиусе N километров получают уведомление.

Прикольная фишка: Интерактивная карта «дружелюбных мест» (гаражи, где пустят погреться/починиться, мото-кафе, проверенные сервисы).

Для команды:

Backend: Работа с геозонами (Geofencing) и Push-уведомлениями.

Frontend: Сложная карта (Leaflet/Mapbox) с кастомными маркерами.

3-й человек: Система чатов внутри заявки и профили техники.
